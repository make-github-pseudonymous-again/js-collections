    js-collections

    Python's collections library for JavaScript. Parent is js-data-structures.

    JavaScript version of Python's collections library together with some of Python's built-in data structures.

    let {
    set , dict ,
    namedtuple ,
    deque ,
    defaultdict ,
    ordereddict ,
    counter ,
    chainmap
} = collections ;
